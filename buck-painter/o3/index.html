BUCK Painter Instructions

Welcome to BUCK Painter!

  • Click and drag on the model to paint using the selected material.
  • Use the right mouse button to rotate, middle / wheel to zoom, and shift-left-drag to pan.
  • Select brush size and material color using the panel on the right.
  • Load your own GLTF/GLB models using the "Load Model" button.
  • Export your painted model using the "Export GLTF" button.
